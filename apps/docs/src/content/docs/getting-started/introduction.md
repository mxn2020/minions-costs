# Minions Costs

Cost tracking, budgets, and financial ledgers for agent operations

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-costs/sdk`) — Core library for Node.js
- **Python SDK** (`minions-costs`) — Python bindings
- **CLI Tool** (`@minions-costs/cli`) — Command-line interface
- **Web App** — Interactive playground at [costs.minions.wtf](https://costs.minions.wtf)
