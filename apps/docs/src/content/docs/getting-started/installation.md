---
title: Installation
description: Install Minions Costs packages
---

## TypeScript / Node.js

```bash
npm install @minions-costs/sdk minions-sdk
```

## Python

```bash
pip install minions-costs
```

## CLI

```bash
npm install -g @minions-costs/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
